## Features 
Lokale Web-Anwendung zum Import von Kamera-Dateien auf eine NAS.
- Session-Erkennung anhand Zeit-Gaps
- Web-GUI im Browser
- Preview fÃ¼r alle Sessions
- Manuelles benennen der Fotosessions
- Import in frei definierbare Zielordner (z. B. NAS)

## Next Steps / Future

## Voraussetzungen
- Node.js >= 18
- macOS oder Linux
